# Hospital-Management-System

 # 🏥 Hospital Management System (SQL + Python + Power BI)
 📌 Overview

  This project is a Hospital Management System that combines SQL, Python (Jupyter Notebook), and Power BI to manage and analyze patient data.

   It demonstrates how raw data can be transformed into meaningful insights using data analysis and visualization tools.

🧰 Tech Stack
🗄️ SQL (Database & Queries)
🐍 Python (Jupyter Notebook for Data Analysis)
📊 Power BI (Data Visualization & Dashboarding)
📂 Database Structure

# The project uses a patients table with the following columns:

1. patient_id – Unique ID
2. name – Patient name
3. age – Age
4. service – Medical service type
5. arrival_date – Admission date
6. departure_date – Discharge date
7. durations – Length of stay
# ⚙️ Features
🔹 SQL Operations
1. Retrieve and filter patient records
2. Aggregate data (COUNT, AVG)
3. Group patients by services
4. Identify active (non-discharged) patients
5. Use of CTE & Window Functions
🔹 Python (Jupyter Notebook)
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Working with datasets using Pandas
4. Generating insights and summaries
🔹 Power BI Dashboard
1. Interactive dashboards
2. Patient distribution by service
3. Age analysis
4. Admission trends
5. Duration-based insights
